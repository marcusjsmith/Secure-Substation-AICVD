Keyboard & Interaction Guide

    

      

        
Canvas

        
Click + drag background
Pan the canvas

        
Scroll wheel
Zoom in / out

        
Fit button
Zoom out to show the entire diagram

        
Center button
Pan so the diagram is centered in the view

        
Shift + drag background
Rubber-band box select


        
Elements

        
Click element
Select it and show properties

        
Shift + click
Add / remove from multi-selection

        
Drag inside element
Move element (or all selected elements)

        
Drag edge / corner handles
Resize element

        
Ctrl/⌘ + A
Select all elements and lines


        
Connections

        
Hover port (blue dot) + drag
Draw a new connection line

        
Drag line endpoint circle
Reconnect to a different port (or detach)

        
Drag curve handle
Adjust the bend of S-shape, Arc, or Elbow lines


        
Copy & Paste

        
Ctrl/⌘ + C
Copy selected elements and their connections

        
Ctrl/⌘ + V
Paste with offset (repeatable)


        
Other

        
Delete / Backspace
Delete all selected elements and lines

        
Esc
Cancel current action or clear selection

        
Save / Open buttons
Save diagram to .json file / reload a saved diagram

        
Export HTML
Generate a standalone SVG snippet of the diagram
